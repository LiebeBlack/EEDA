# EEDA
ÉEDA no es solo una aplicación; es un ecosistema pedagógico diseñado para transformar la educación en ciberseguridad. Basado en el modelo CPA (Concreto-Pictórico-Abstracto) y principios de gamificación, el sistema permite a usuarios de 3 a 16 años navegar conceptos de alta complejidad mediante metáforas visuales.
