*"E.E.D.A. (Ecosistema Educativo Digital Adaptable)"*

E.E.D.A. no es solo una aplicación; es un ecosistema pedagógico diseñado para transformar la educación en ciberseguridad. Basado en el modelo CPA (Concreto-Pictórico-Abstracto) y en principios de gamificación, el sistema permite a usuarios de 3 a 16 años navegar por conceptos de alta complejidad mediante el uso de metáforas visuales.
