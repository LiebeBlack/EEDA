Información de Telemetría

Infraestructura

El sistema E.E.D.A opera sobre una arquitectura serverless distribuida, utilizando GitHub Pages para el frontend y GitHub Actions para la automatización de métricas.

  • Hosting: GitHub Pages
  • CI/CD: GitHub Actions
  • Datos: JSON estático + API GitHub
  • Actualización: Automática cada 24h

Métricas Trackeadas

Nuestro sistema recopila y analiza múltiples dimensiones del desarrollo del proyecto para mantener una visión integral del progreso.

  • Conteo de commits
  • Issues abiertos/cerrados
  • Pull requests
  • Contribuidores activos

Seguridad & Privacidad

Todos los datos de telemetría se procesan localmente y se transmiten mediante conexiones seguras. No almacenamos datos personales de usuarios.

  • Conexiones HTTPS forzadas
  • Tokens de API seguros
  • Sin tracking de usuarios
  • Código open source
