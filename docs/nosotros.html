Sobre el Proyecto

E.E.D.A es una iniciativa de software educativo desarrollada con enfoque en accesibilidad, rendimiento y soberanía técnica.

Níkolas Gómez (Liebe Black)

Software Architect & Fullstack Writer

GitHub Profile Portafolio Web

Desarrollo Regional

Proyecto originado en la Isla de Margarita, Venezuela, orientado a cerrar brechas digitales mediante tecnología adaptada a contextos locales con recursos limitados.

Optimización de Recursos

Software diseñado para funcionar eficientemente en hardware de gama media, eliminando componentes innecesarios que consuman recursos del sistema.

Impacto Social

Herramientas de calidad profesional disponibles para educadores y estudiantes, con especial atención a necesidades de accesibilidad y diversidad cognitiva.

El proyecto se fundamenta en principios de excelencia técnica, diseño centrado en el usuario y compromiso con la educación inclusiva.
